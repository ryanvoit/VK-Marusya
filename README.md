# VK-Marusya

Данное приложение реализовано с нуля, используя **React** и **Next.js**. Также в разработке использовались библиотеки: **Zod**, **Antd**, **React-query**, **Axios**. Также использовался фреймворк **Tailwind**.

## Функионал

В этом приложении пользователи смогут искать фильмы. На данный момент в нем есть две страницы: "Главная" и "Жанры". На странице 'Главная' Вы можете увидеть рандомный фильм из огромного каталога фильма. Также на этой странице представлены 10 фильмов имеющие рейтинг равный 10 по IMDb-рейтингу. На странице "Жанры" фильмы будут распределены по разным жанрам.

## Запуск приложения

Для того, чтобы запустить приложение надо:

1) Открыть проект и запустить терминал
2) Установить необходимые зависимости, для этого в терминале надо прописать:
```sh
npm install
```
3) Запустить приложение, для этого в терминале надо прописать:
```sh
npm run dev
```
4) После запустка в терминале появится строка с ссылкой по которой надо перейти:

▲ Next.js 16.2.9 (Turbopack)
- Local:         http://localhost:3000
